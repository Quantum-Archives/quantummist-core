# 📂 FOLDER_STRUCTURE.md – Struktura serwera QuantumMist

> Bazowa struktura projektu dla `/srv/quantummist/` na serwerze produkcyjnym ND5

---

## 🔩 Root projektu `/srv/quantummist/`
```
quantummist/
├── admin/                 # Pliki panelu Ishkarim Admin + konfiguracje
│   ├── uploads/           # (w przyszłości) katalog wrzutek plików
│   ├── panel/             # Pliki HTML panelu admina
│   ├── config/            # sections_index.json, sloty, style
│   └── logs/              # logi wrzutek, zdarzeń admina
├── aerendal/              # Archiwum osobiste ND5
│   ├── teksty/            # Pliki .md
│   └── aerendal_index.json
├── sections/              # Treści systemowe ND5 (tarot, horoskop...)
│   ├── tarot/
│   │   ├── zadaniowiec/
│   │   ├── wrazliwy/
│   │   └── opiekun/
│   ├── horoskop/
│   └── numerologia/
├── scripts/               # render.js, menu.js, admin.js
├── static/                # CSS, JS, img
├── templates/             # section_view.html, aerendal_view.html
├── users/                 # folder użytkowników (dzienniki, zapisy)
├── main.py                # Backend Flask API
├── README.md              # Opis startowy projektu
└── setup.sh               # Skrypt konfiguracyjny dla serwera (instalacja, porty)
```

---

## 🔐 Uwaga
- `sections/` i `aerendal/` są **dynamicznie ładowane** – zmiany plików nie wymagają restartu serwera
- `admin/config/` powinien być kontrolowany tylko przez ND5 (Ciebie)
- `setup.sh` zawiera instrukcje do instalacji zależności, struktury katalogów, bibliotek i launchera serwera
