# 📓 ECHO_SLOW_JOURNAL_LOGIC.md – Panel Dziennika Echo Słów

## 🎯 Cel
Zapewnić użytkownikowi prywatną przestrzeń do edycji, katalogowania i przetwarzania otrzymanych (lub samodzielnie utworzonych) kanw tarotowych, notatek i refleksji.

---

## 📐 Layout interfejsu (desktop)

| Element                       | Wymiar       | CSS px (96 DPI) |
|------------------------------|--------------|------------------|
| Lista segregatorów / kanw    | 6 cm × 14 cm | ~230px × 530px |
| Okno edytora tekstowego      | 14 cm × 14 cm| ~530px × 530px |
| Pasek nagłówka (tytuł / meta)| 1 cm wysokości| ~38px |

---

## 🧱 Główne komponenty

### `slot:journal_sidebar`
- Lista segregatorów + lista wpisów
- Możliwość tworzenia nowych segregatorów
- Drag'n'drop lub dodawanie wpisu do segregatora

### `slot:journal_editor_main`
- Edytor WYSIWYG z podstawowymi opcjami tekstowymi
- Widok bieżącej kanwy lub nowego wpisu
- Tytuł, tagi, opis, opcjonalnie załącznik graficzny (później: prompt do generacji)

### `slot:journal_entry_metadata`
- Meta: data utworzenia, źródło (czat, tarot, manualny wpis)
- Ikona, typ, kolor przypisany do stylu wpisu

---

## ✏️ Podstawowe funkcje edytora (WYSIWYG)

| Funkcja | Opis |
|--------|------|
| 🅰️ Akapit | Enter → nowy akapit (autozapis) |
| 🔤 Pogrubienie | `Ctrl+B` lub przycisk „B” |
| 🔠 Kursywa | `Ctrl+I` lub przycisk „I” |
| 🔽 Podkreślenie | `Ctrl+U` lub przycisk „U” |
| 📋 Lista punktowana | wstawia `<ul>` z elementami |
| 1️⃣ Lista numerowana | `<ol>` z numeracją |
| 🔗 Wstawienie linku | `Ctrl+K` lub ikona 🔗 |
| 📎 Załącz plik/grafikę | tylko dla zaufanych userów |
| 🎨 Styl bloków | cytat, kod, nagłówek (H1–H3), separator |

### 🧠 Opcje rozszerzalne (później)
- 🎴 Przycisk „Wygeneruj grafikę z tekstu” (dla subów)
- 📌 „Zablokuj wpis” (do archiwizacji)
- 🧬 „Scal wpis z innym” (tworzenie większych rozdziałów)
- 🔄 Historia wersji wpisu (revert)

## 🔌 Sloty systemowe
- `slot:journal_sidebar` ← nawigacja po segregatorach / wpisach
- `slot:journal_editor_main` ← główny edytor tekstowy
- `slot:journal_entry_metadata` ← pasek z info
- `slot:journal_submit` ← opcje zapisania, eksportu, wyślij do AI
